# Examples

#Browser playground

This example shows how to work with casperapi in serverless environment This example contains

  • Saving
  • Updating
  • Downloading
  • Generating a temporary link to the file
  • Displaying media (like video \ audio \ images)

Please note that barely any transpiling and polyfilling is done for that example, so consider using latest Chrome \ Firefox \ Edge. Video playback from a timestamp support will be implemented later.
